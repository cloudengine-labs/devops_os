🚀 DevOps-OS

Automate your entire DevOps lifecycle — from CI/CD pipelines to Kubernetes deployments and SRE dashboards.

🎉 Announcing the DevOps-OS Launch

We are thrilled to announce the official launch of DevOps-OS — an open-source DevOps automation platform that scaffolds production-ready CI/CD pipelines, Kubernetes configurations, and SRE observability configs in seconds, from a single CLI command or an AI chat prompt.

✨ Key Features

🚀 CI/CD Generators

One-command scaffolding for GitHub Actions, GitLab CI, and Jenkins pipelines — production-ready from day one.

☸️ GitOps Config Generator

Generate Kubernetes manifests, ArgoCD Applications, and Flux CD Kustomizations with a single command.

📊 SRE Config Generator

Create Prometheus alert rules, Grafana dashboards, and SLO manifests aligned with your reliability targets.

🤖 MCP Server

Plug DevOps-OS tools directly into Claude or ChatGPT as native AI skills for conversational DevOps automation.

🛠️ Dev Container

Pre-configured multi-language development environment supporting Python, Java, Go, and JavaScript out of the box.

🔄 Process-First Philosophy

Built-in education on the Process-First SDLC philosophy and how it maps to every DevOps-OS tool and workflow.

🏗️ Tech Stack

DevOps-OS integrates with industry-leading tools across the DevOps lifecycle:

  • CI/CD: GitHub Actions, GitLab CI, Jenkins
  • GitOps: ArgoCD, Flux CD
  • Infrastructure: Kubernetes, Helm, Kustomize
  • Observability: Prometheus, Grafana
  • AI Integration: MCP Server (Claude, ChatGPT)

🚦 Getting Started

Clone the repository and scaffold your first pipeline in under a minute:

git clone https://github.com/cloudengine-labs/devops_os.git
cd devops_os
pip install -r cli/requirements.txt

# Check version
python -m cli.devopsos --version           # → devopsos version 0.2.0

# Scaffold a complete GitHub Actions workflow
python -m cli.devopsos scaffold gha --name my-app --languages python --type complete
